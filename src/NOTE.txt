===== IntelliJ Mac Shortcuts

Ctrl+R : Run
Ctrl+D : Debug

Cmd+X : 行切り取り
F1 : Javadoc Lookup

Alt+Cmd+V : Extract as a local variable
Alt+Cmd+F : Extract as a field

カーソル移動
Alt + -> : Move Caret to Next Word
Cmd + -> : Move Caret to Line End
Cmd + <- : Move Caret to Line Start
Ctrl + 上下 : 次のメソッドに移動
Cmd + Alt + [ : 現在のブロックの先頭に移動
Cmd + Alt + [ : 現在のブロックの末尾に移動

===== IntelliJ Windows Shortcuts

Shift+F10 : Run
Shift+F9 : Debug

Cmd+X : 行切り取り

Alt+Ctrl+V : Extract as a local variable
Alt+Ctrl+F : Extract as a field

Ctrl+Shift+V : コピー履歴から貼り付け
